Guide to website

Introduction

Hello! My name is Michael Alan Murphy Quille (you can call me Alan). This is a website for the Data Representation and Querying Project 2020, Option A. In this web server, you can perform basic CRUD operations on two SQL tables in a database called datarepresentation called student and lecturer. You can only perform operations on one at a time (see below on how to change the current table). The buttons above take you to each page in this web server. The functionality of each page is described in the following sections. There is also some extra API functionality at this link. This inner joins the student and lecturer table if they are both present in the database. In addition, links to create and delete the student and lecturer tables are available here:

Home

This button takes you to this page, the homepage.

Create

This button takes you to a page that allows one to create rows in specified SQL tables.

Read

This button takes you to a page that allows one to view a row from a SQL table or all rows from a SQL table

Update

This button takes you to a page that allows one to change a row from a SQL table.

Delete

This button takes you to a page that allows one to delete a row from a SQL table.

Gmail

This button takes you to a page that allows one to load in subject lines for the current date from an email address called datarepresentation2020@gmail.com to a SQL table

Logout

This button takes you to a page that logs you out of the page and returns you to login.html

Change table

This button takes you to a page that changes the SQL table from student to lecturer (and vice versa) and returns you to this page.
